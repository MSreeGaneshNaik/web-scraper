What it extracts

Per page, plus site-wide intelligence — all in one crawl.

📝

Content

Title, meta tags, all headings, paragraph text, list items & full word count.

🔗

Links & images

Every link (internal/external) with anchor text, plus every image with alt & title.

📊

Tables & forms

Tables as rows of cells, and forms with action, method & fields (incl. hidden values).

📇

Contacts

Emails, phone numbers & social profiles — exported to a tidy contacts CSV.

🛠️

Tech fingerprint

Detects WordPress, React, Shopify, jQuery, Cloudflare and more.

🗺️

Sitemaps & robots

Reads robots.txt + sitemap index to find pages that aren't linked anywhere.

👻

Hidden extras

HTML comments, data-* attributes, hreflang, RSS/Atom feeds, favicon & manifest.

✨

Deep scraping

Opt-in: render JS with a headless browser, capture API calls, take screenshots & read PDFs/Word.

Run it in 30 seconds

It's a small Flask app — clone, install, run.

# clone & enter
git clone https://github.com/MSreeGaneshNaik/web-scraper.git
cd web-scraper

# install & launch
pip install -r requirements.txt
python app.py

# then open
http://127.0.0.1:8000
